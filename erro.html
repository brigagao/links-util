PILHA

Conceito

Quando trabalhamos com Arrays é muito importante conhecermos o conceito de pilhas.

Na programação, as pilhas são estruturas que armazenam elementos em uma ordem específica, no caso das pilhas os últimos são os primeiros..

O termo correto para este conceito é o LIFO que em inglês significa Last In First Out.

Para se trabalhar com pilhas usamos dois comandos. O array_push que insere um valor no final de uma fila e o array_pop que remove o último elemento de uma pilha

Um exemplo que podemo usar é o de um empilhamento de livros, onde o último livro a ser colocado é o primeiro a ser removido, pois esta no top.

Observe que os elementos são colocados em ordem seqüencial e retirados do final (último elemento inserido) até o início.

Os retornos destes comandos são:

array_push: Retorna o novo número do elemento do array;

array_pop: Retorna o último elemento do array, eleminado-o o mesmo.
